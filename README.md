StyleAI – AI Powered Fashion Styling Advisor (Project Description)

StyleAI is an intelligent fashion styling platform that uses Artificial Intelligence to provide personalized fashion recommendations. The system uses the Groq API with the LLaMA-3.3-70B model to analyze user images and generate styling suggestions based on their skin tone, gender, and fashion trends.

The main goal of this project is to help users choose clothes, colors, accessories, and hairstyles that suit them best. By uploading a facial photograph, the system performs skin tone detection and provides customized fashion advice.

The platform combines image processing techniques and AI analysis to deliver fast and accurate styling suggestions. It uses OpenCV and Pillow libraries to process images and detect skin tone categories such as Fair, Medium, Olive, and Deep. Based on the detected skin tone, the AI model generates fashion recommendations that match the user's appearance.

StyleAI also provides a curated shopping experience by suggesting products from popular online shopping platforms such as Amazon, Myntra, and Zara. These product recommendations include clothing items like shirts, pants, shoes, and accessories that match the user’s styling profile.

The system is built using a Flask-based backend and a modern web interface developed with HTML, CSS, and JavaScript. The backend processes image uploads, performs skin tone analysis, and communicates with the Groq AI model to generate styling insights. The frontend provides a user-friendly interface for uploading images, selecting gender, and viewing styling results.

StyleAI improves the traditional fashion recommendation process by making it automated, personalized, and accessible to everyone. It demonstrates how artificial intelligence can be used in the fashion industry to provide smart styling assistance while maintaining high performance and accuracy.
